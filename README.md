# Gaming Player Retention Funnel Analysis
Player retention funnel analysis using Riot Games API, PySpark, SQL, and Power BI


## Project Overview
Analysis of player progression and retention in competitive gaming to identify drop-off points and opportunities to improve player engagement. This project demonstrates end-to-end data analytics skills from API data collection to business insights visualization.

## Business Problem
Gaming companies lose significant revenue when players quit early. This analysis tracks players through key progression milestones to identify:
- Where players drop off in their journey
- What factors predict long-term retention
- Actionable recommendations to improve player engagement
